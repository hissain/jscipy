jSciPy

A Java library for scientific computing, inspired by SciPy.

View Javadoc View on GitHub

Overview

jSciPy fills the gap in the Java ecosystem for a comprehensive scientific computing library similar to Python's SciPy. It provides robust implementations for signal processing, optimization, and more.

Features

  • Signal Processing: Butterworth filters (including zero-phase filtfilt), Peak Detection, Savitzky-Golay filters, Median filters, Convolution, Detrending, Resampling.
  • Math & Analysis: RK4 ODE Solver, Interpolation (Linear, Cubic Spline), FFT/IFFT (via Commons Math wrapper with enhanced utilities), Hilbert Transform.

Getting Started

Add the dependency via JitPack:

repositories {
    mavenCentral()
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.hissain:jSciPy:2.1.0'
}
